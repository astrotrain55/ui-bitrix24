Component library in Bitrix24 style

## Install

```nodejs
npm i -S vue-bitrix24@latest bitrix24-library@latest
```

## Use

```vue
// App.vue
<template>
  <bx-button menu>bx-button</bx-button>
</template>

<script setup lang="ts">
import { BxButton } from 'vue-bitrix24';
</script>
```
